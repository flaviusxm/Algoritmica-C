O aplicatie de tip consola care are ca scop invatarea si exemplificarea algoritmilor fundamentali din informatica prin implementari practice in limbajul C.

**Cum rulez ?**

Ruleaza in folderul proiectului `make` pentru Linux/Unix/Mac  
sau ruleaza `mingw32-make` (pentru pachetul mingw) daca esti pe Windows / sau direct compile_files.bat 

Dupa compilare, ruleaza `main_exec.exe` pe Windows sau `./main_exec` pe Linux/Mac.
